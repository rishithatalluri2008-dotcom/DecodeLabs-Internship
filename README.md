# Random Password Generator

## Description
A Python project that generates secure random passwords based on the user-specified length.

## Features
- User-defined password length
- Includes uppercase and lowercase letters
- Includes digits
- Includes special characters
- Ensures password strength
- Input validation

## Technologies Used
- Python 3
- random module
- string module

## How to Run
1. Open terminal.
2. Run:
   python password_generator.py
3. Enter the required password length.
4. A secure password will be generated.
